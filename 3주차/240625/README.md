# 리액트

- 사용자 인터페이스를 만들기 위한 자바스크립트 라이브러리
- 라이브러리 : 자주 사용되는 기능을 정리해 모아 놓은 것이라는 의미
- 사용자 인터페이스 (UI, User Interface)
  - 사용자와 컴퓨터 프로그램이 서로 상호작용을 하기 위해 중간에서 서로 간에 입력과 출력을 제어해주는 것
  - 웹사이트 => 버튼, 텍스트 입력 폼 등
  - UI 라이브러리 : 사용자 인터페이스를 만들기 위한 기능 모음집(라이브러리)
